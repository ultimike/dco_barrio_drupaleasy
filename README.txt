Example DrupalEasy theme based on Bootstrap 5, the Drupal Bootstrap Barrio
base theme, and "Bootstrap 4 - Barrio SASS Starter Kit".

The node_modules/ and package-lock.json files are both ignored from this
repository, so be sure to run "npm install", if necessary.

This example is part of Drupal Career Online, a 12-week Drupal training program.
More information can be found at https://www.drupaleasy.com/dco
